## Instalasi

1. Clone repository

bash
git clone https://github.com/kevinssheva/tugas-1-kriptografi-koding.git

2. Install npm package

bash
npm install

3. Run development

bash
npm run dev

4. Open website

bash
http://localhost:5173/
